

- `alt` + `.` （或者用 `Ecs` + `.` ）：打印上一条命令的参数


- `ctrl` + `a`：移动至开头
- `ctrl` + `e`：移动至结尾
- `alt` + `f`：向前移动一个单词
- `alt` + `b`：想后移动一个单词


- `ctrl` + `u`：删除光标左侧全部字符
- `ctrl` + `k`：删除光标右侧全部字符
- `ctrl` + `w`：删除光标左侧单词
- `alt` + `d`：删除光标右侧单词
- `ctrl` + `d`：删除光标右侧一个字符


- `ctrl` + `l` ：相当于执行 `clear`
- `ctrl` + `r` ：搜索命令
