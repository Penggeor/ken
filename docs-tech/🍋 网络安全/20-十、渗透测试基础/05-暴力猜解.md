## 安全防范

1️⃣ 设计安全的验证码

2️⃣ 对验证码的有效期和次数进行验证

3️⃣ 限制错误认证，比如连续 5 次密码错误锁定两小时

4️⃣ 双因子验证，利用 token 防止暴力破解
